# hddl_project_2025
Project about dogs and cats dataset is stored in folder **Code**
## Code/utils/dataset.py
This file contains the dataset class and custom collate functions for loading and processing the cat and dog images, annotations, and masks.
## Code/utils/train_val.py
This file contains training functions for different models including cat/dog classifier, breed classifier, U-Net segmentation model, and a multitask model combining classification and segmentation.
## Code/models/
This folder contains the model architectures used in the project.
## Code/train.py
This file is used to train the models using the training functions defined in `train_val.py`.
## Code/evaluate.py
This file is used to evaluate the performance of the trained models on validation or test datasets.
